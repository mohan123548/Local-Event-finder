# 🧑‍🤝‍🧑 USER PERSONAS & USAGE SCENARIOS
## 👤 Persona 1 — Alex
	
Age	21
Role / Occupation	University Student
Goals	Find nearby social events, workshops, and student gatherings
Frustrations	Misses events because information is scattered across different platforms
Tech Comfort	High digital literacy and active social media user
📌 Usage Scenario

Alex is a university student who enjoys participating in workshops, college festivals, and social meetups. Due to the large number of platforms sharing event information, he often misses important events happening around him. Using the Local Event Finder application, Alex can quickly explore nearby student events based on category, location, and date. The application allows him to save favorite events, receive event reminders, and stay connected with campus activities in one centralized platform.

✅ Expected Benefits
Easy access to student and social events
Saves time by avoiding multiple platforms
Personalized event recommendations and reminders
## 👤 Persona 2 — Sarah
	
Age	25
Role / Occupation	Working Professional
Goals	Discover weekend events and networking opportunities
Frustrations	Limited time to search multiple websites for events
Tech Comfort	Comfortable using mobile and web applications
📌 Usage Scenario

Sarah is a busy working professional who wants to make productive use of her weekends by attending networking sessions, workshops, and entertainment events. However, searching across different websites consumes too much time. With the Local Event Finder application, Sarah can filter events based on her interests, preferred location, and schedule. The platform helps her discover suitable events quickly and provides complete event details in a simple and user-friendly interface.

✅ Expected Benefits
Quick and efficient event discovery
Centralized access to event information
Improved work-life balance through better planning
## 👤 Persona 3 — David
	
Age	28
Role / Occupation	Tourist
Goals	Explore cultural and entertainment events in the city
Frustrations	Difficulty finding updated local event information
Tech Comfort	Moderate digital literacy
📌 Usage Scenario

David is a tourist visiting a new city and wants to experience local culture, entertainment, and community activities. Since he is unfamiliar with the city, finding accurate and updated event information becomes challenging. By using the Local Event Finder application, David can discover nearby cultural festivals, live events, exhibitions, and entertainment programs based on his current location. The application also provides venue details, event timings, and directions, helping him enjoy a better travel experience.

✅ Expected Benefits
Real-time access to local events
Better travel and cultural exploration experience
Easy navigation and location-based event suggestions
## 🎯 Conclusion

The Local Event Finder application is designed to simplify event discovery for different types of users, including students, professionals, and tourists. By providing a centralized platform for browsing, filtering, and accessing local event information, the application improves user engagement, saves time, and enhances the overall event participation experience.
